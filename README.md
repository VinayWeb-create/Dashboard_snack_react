VendorHub – A full stack vendor management dashboard for restaurants.

Features:
- Vendor authentication (JWT)
- Product management (CRUD operations)
- Analytics dashboard
- Firm/profile management
- Responsive UI

Tech Stack:
React, Node.js, Express, MongoDB, Cloudinary

Live Demo:
https://dashboard-snack-react.vercel.app/
